Formation : Développeur WEB
Projet 2 : RESERVIA - un outil de planification des vacances
Intégration de la maquette responsive du site Reservia en HTML et CSS

Éléments fournis

Maquettes : desktop et mobile (mais le site doit être adapté à l'affichage sur tablette également)
Images et logos : différents formats pour bien adapter le choix de la taille
Icônes : provenance de Font Awesome
Charte graphique : bleu #0065FC, bleu claire #DEEBFF, fond gris #F2F2F2
Police : Raleway (Google Fonts)


Fonctionnalités

-L'outil doit permettre aux usagers de rechercher des hébergements dans la ville de leur choix, en le saisissant dans le champ de recherche. Le bouton de recherche n'est pas fonctionnel à ce stade.
-Chaque carte d’hébergement ou d’activité est cliquable dans son intégralité. En revanche, les liens sont vides pour l'instant.
-Cette version intégre les filtres qui ne sont pas fonctionnels pour le moment, mais ils changent d’apparence au survol.
-Dans le menu, les liens “Hébergements” et “Activités” sont des ancres qui menent aux sections de la page.

Contraintes techniques

Utilisation de Framework ou pré-compilateur non recommandé pour ce projet
Utilisation de Visual Studio Code avec les plugins Live Server et Prettier
Utilisation des Flexbox est fortement recommandé, mais possibilité d'utiliser Grid CSS également
Les fichiers HTML et CSS doivent être séparés et les dossiers bien orgnanisés
Utilisation indispensable des balises sémantiques
Le code ne doit pas comporter des erreurs, ni alertes et doit etre validé avec validateur W3C
Le site doit être compatible avec les dernières versions de Firefox et Chrome
Versionnage doit être effectuer avec GitHub et cela dès le début. Le site doit être hébérgé sur GitHub Pages
